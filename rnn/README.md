# RNN (Siamese GRU) for Quora Question Pairs

Duplicate question detection using a bidirectional GRU siamese network on Qwen3-Embedding-4B embeddings (2560-dim).

## Results

| Metric | Score |
|--------|-------|
| Accuracy | 87.76% |
| F1 | 84.62% |
| Precision | 78.92% |
| Recall | 91.20% |

Training time: 37s on NVIDIA H200 NVL. See `results/metrics.txt` for the full classification report and `results/config.json` for all hyperparameters.

## Architecture

Each 2560-dim embedding is split into 10 chunks of 256, fed through a shared 2-layer bidirectional GRU (hidden_size=128). The two question representations are combined via [h1; h2; |h1-h2|; h1*h2] and passed through a classification head. Total parameters: 871,809.

## How to reproduce

Requires `embeddings.zarr` (generated by `embed_quora.py` using Qwen3-Embedding-4B).

```bash
# On the SOC cluster (SSH into xlogin.comp.nus.edu.sg)
# 1. Generate embeddings (if not already done)
sbatch slurm_gpu.sh embed_quora.py

# 2. Train the RNN
sbatch slurm_gpu.sh rnn/train.py --epochs 10
```
